## 2. Description of Service

Masqueroll is a Discord bot that provides dice rolling utilities for Vampire: The Masquerade 5th Edition, including character sheet management, macros, and roll tracking.

## 3. User Data

Masqueroll does not maintain its own database. All character sheet data (stats, name, image, macros) is stored as messages in a dedicated `#character-sheets` channel within your Discord server. The bot reads and edits those messages to provide its functionality.

Data is therefore subject to Discord's own data policies. You can delete your data at any time using the `/delete_character` and `/remove_macro` commands, which will delete or edit the corresponding Discord messages.
